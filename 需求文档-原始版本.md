# HIRS 脊柱侧弯智能康复辅助系统 - 需求文档

**版本**: v1.0  
**日期**: 2025-03-11  
**项目名称**: HIRS (Home-based Intelligent Rehabilitation System for Scoliosis)

---

## 1. 项目概述

### 1.1 项目背景

脊柱侧弯是一种常见的脊柱畸形疾病，需要长期、规范的康复训练。传统的康复训练存在以下问题：

- 患者居家训练缺乏专业指导
- 康复师难以跟踪患者训练依从性
- 训练效果缺乏数据化评估
- 复诊时缺乏完整的训练历史记录

### 1.2 项目目标

HIRS系统旨在通过智能化技术，为脊柱侧弯患者提供专业、便捷的居家康复训练服务，同时为康复师提供高效的患者管理工具。

### 1.3 核心价值

- **标准化**: PSSE训练方案的标准化管理
- **可视化**: 训练动作的视频/动画演示
- **计时化**: 精准的训练计时和进度控制
- **反馈化**: 实时训练反馈和异常记录
- **数据化**: 完整的训练数据记录和统计分析

## 2. 系统架构

### 2.1 系统组成

HIRS系统由两个端组成：

1. **康复师端** (HIRS Therapist)
   - 面向康复科医生、物理治疗师
   - 平台: iOS (iPad优先) + Android
   - 功能: 患者管理、训练计划制定、数据查看

2. **患者端** (HIRS Patient)
   - 面向脊柱侧弯患者
   - 平台: iOS + Android (手机APP)
   - 功能: 居家训练、进度跟踪、数据上传

## 6. 非功能性需求

### 6.1 性能要求

- 应用启动时间 < 3秒
- 页面切换响应时间 < 500ms
- 训练计时精度误差 < 100ms
- 支持离线使用核心功能

### 6.2 安全要求

- 符合医疗数据隐私保护法规
- 用户数据加密存储
- JWT Token认证
- 传输加密（HTTPS）
- 数据备份机制

### 6.3 可用性要求

- 支持中/英双语
- 界面简洁直观
- 符合无障碍设计规范
- 最小触摸目标 44x44px

### 6.4 兼容性要求

- iOS 13.0+
- Android 8.0+
- 支持手机和平板
